Getting Started
A terminal file reader that renders markdown, diffs, images, and transcripts inline.
Installation
bash
brew install wildreason/tap/aster
alias as=aster
Usage
Open any file and aster picks the right renderer:
CommandWhat it does
as file.mdRendered markdown with colors
as photo.pngImage inline in terminal
as changes.diffSyntax-highlighted diff
as data.jsonlTranscript viewer
as pickChoose from recent files
Navigation
Scroll through documents with familiar keybindings:
- j / k scroll down and up
- d / u half-page jumps
- g / G top and bottom
- q quit
Read more at github.com/wildreason/reader
