## 1. CSV/TSV parser

type: feature | priority: p1 | depends: none | status: done

### Criteria

- `CsvParser` implements `Parser` interface (Parse + Detect)
- Detects .csv and .tsv extensions
- Parses headers from first row
- Handles quoted fields with commas, newlines, and escaped quotes
- Uses Go stdlib `encoding/csv` (no external deps)
- TSV handled via tab delimiter detection
- Returns single Block with ContentType BlockContentCSV
- Stdin auto-detection: content with consistent comma-separated columns detected as CSV

## 4. Auto-chart detection

type: feature | priority: p2 | depends: 3 | status: done

### Criteria

- When CSV has a plausible x-axis (dates, sequential labels) and numeric y columns, show a chart
- Chart renders above the table as a simple SVG (no charting library)
- Supports: line chart (time series), bar chart (categorical)
- Chart is optional -- only appears when data shape fits
- No chart for non-numeric data or single-column CSVs
- Chart uses brand colors (Accent Blue #3B82F6 primary, Slate #1E293B secondary)
